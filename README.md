## 🚀 **About**
<span style="font-size: 2.5em; font-weight: bold; color: #58A6FF; display: inline-block; animation: glow 2s ease-in-out infinite alternate;">
  About Me
</span>

<img align="right" width="360" src="https://media.giphy.com/media/qgQUggAC3Pfv687qPC/giphy.gif" alt="Coding animation"/>

I'm a full-stack developer who cares equally about how systems are built and how they feel to use. I work across the entire stack — from crafting interfaces with precise attention to motion and layout, to designing backend systems that are fast, maintainable, and built to last.

My work tends to sit at the intersection of clean architecture and product thinking. I like code that reads like prose and systems that behave predictably under pressure.

When I'm not shipping, I'm exploring compilers, contributing to open source, and slowly convincing myself that Neovim was worth the learning curve.
